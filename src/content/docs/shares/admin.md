Every share has an admin page at **Shares → [the share]** where staff can review what a family submitted.

## What you can do

- See **share details**: title, type, recipient, creation date, expiration, use count, PIN lock state.
- View each **form submission** with field labels next to the parent's answers, including image previews.
- View each **submitted document** and print it, or click **Print All** to print every submitted PDF in one dialog.
- **Approve** the packet to run all automated actions (shown only when the packet is in [Requires Approval](/packets/approval-mode/) mode and all steps are complete).
- **Revoke** the share to permanently delete it.

## The "Needs Approval" badge

This badge appears when:

- The share is a packet, **and**
- Every step has been submitted, **and**
- The packet is in **Requires Approval** mode, **and**
- No actions have run yet.

Clicking **Approve** runs all actions for the packet at once.
