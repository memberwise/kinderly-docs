## Multiple boards

You can have more than one board — for example, one for **Infant/Toddler leads** and one for **Summer Camp leads**. Multiple boards require a **Pro** plan.

- Switch between boards using the **chevron ( < > )** buttons in the page header.
- Open the **Board Manager** dialog to rename or delete boards. (You cannot delete the last board.)
- Create a board from a **template** to start with a predefined column structure.

## Columns

- Click **+ Add Column** to add a new stage.
- Click a column's name to rename it.
- Drag a column header sideways to reorder.
- Delete a column from its menu. (Deleting a column deletes every card in it.)
