A **packet** is a step-by-step onboarding flow you send to a family. Each step must be finished before the next one unlocks. A packet can combine:

- **Forms** for parents to fill in.
- **Documents** for parents to review and sign.
- **Actions** that run automatically when a step is finished (e.g., create a Grow card, send an email).
- **Conditions** that branch the flow based on what a parent answered.

## When to use a packet vs. a single form

- **Single form / document share** — one-off paperwork, a single permission slip.
- **Packet** — full enrollment, returning-family annual paperwork, anything with multiple steps.
