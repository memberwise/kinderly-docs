---
title: Using form answers in actions
description: Pass information a parent just entered into an email or Grow card.
sidebar:
  order: 2
---

When you configure an action, each field can be either:

- **Typed text** — a fixed value (e.g., `enrollment@yourcenter.com`).
- **Answer placeholder** — a reference to a form answer. Pick any field from an earlier form step in the packet, and Kinderly will fill in what the parent entered.

This lets you do things like:

- Send the **welcome email** to the address the parent just typed in the enrollment form.
- Create a **Grow card** titled with the child's name from the form.
