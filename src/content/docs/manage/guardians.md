The **Guardians** area (`/manage/guardians`) stores a record for each parent or guardian. A guardian can be linked to one or more children.

## What you can track

Name, relationship, employer info, billing accounts, custom fields, addresses, notes, app access, and documents.

Detail tabs:

- **Info**
- **Employment**
- **Children** — every child this guardian is linked to
- **Billing** — the guardian's billing accounts
- **Addresses**
- **Notes**
- **Custom Fields**
- **App Access** — the guardian's Parent app login
- **Documents**

## New Guardian Wizard

Create a guardian and link them to one or more children in a single multi-step dialog.

## App Access

Give a guardian a login for the **Parent mobile app**:

- Send an invite.
- Reset the password.
- Block or unblock access.
