title: Field types
description: Every kind of field you can add to a form.
sidebar:
  order: 3
---

| Type | What parents see |
|---|---|
| Text | A single-line box for short answers (name, ID number). |
| Email | An email address box. |
| Phone | A phone number box. |
| URL | A web address box. |
| Date / time | A calendar and clock picker. |
| Checkbox | A single yes/no toggle. |
| Choice | A set of radio buttons — pick one. |
| Dropdown | A dropdown — pick one. |
| Signature | A box parents sign with finger or mouse. |
| Image | A picture you display inline (e.g., a diagram). |
| File upload | Lets parents upload a JPG or PNG (up to 5 MB). |
| Legal | A block of consent text parents must acknowledge. |
| Payment | Placeholder for future in-form payment collection. |
| Separator | A horizontal divider. |
| Label | A bold heading. |
| Description | A longer rich-text block (heading + body). |
| Hidden | Carries a preset value behind the scenes; parents don't see it. |
