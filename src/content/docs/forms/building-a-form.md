1. Go to **Forms** in the left navigation.
2. Click **New Form**, give it a name and (optional) description.
3. Drag fields from the palette on the left onto the canvas.
4. Click any field to edit its **label**, **placeholder**, **hint**, and whether it is **required**.
5. Reorder fields by dragging.
6. Use **Preview** to see exactly what the parent will see.
7. Save.

## Field options you can set

Each field supports:

- **Label**, **placeholder**, and **hint text**
- **Required / optional**
- **Disabled**
- **Minimum and maximum length** (for text)
- **Default value**
- **Choices** (for dropdown and radio-button fields)
