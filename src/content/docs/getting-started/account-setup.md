When you sign up for Kinderly, we automatically create a team for you and set it up on the **free** plan. From there:

1. **Invite your staff.** _(Describe how to invite teammates.)_
2. **Set up billing.** See [Billing → Plans](/billing/plans/) for when and how to upgrade to Pro.
3. **Turn on Kinderly Manage** if you want the full CCMS features — children, staff, classrooms, attendance, and family billing. See [Kinderly Manage](/manage/overview/).
4. **Build your first form** — see [Forms → Building a form](/forms/building-a-form/).
