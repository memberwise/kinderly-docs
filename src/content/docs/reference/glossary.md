| Term | Meaning |
|---|---|
| **Form** | A reusable set of fields a parent fills in. |
| **Document** | A PDF in your team's library. |
| **Packet** | A multi-step onboarding flow combining forms, documents, actions, and conditions. |
| **Action** | An automated task that runs when a packet step is completed (e.g., send email, add Grow card). |
| **Condition** | A logic node inside a packet that branches the flow based on a parent's answer. |
| **Share** | A PIN-protected link you send to a family to deliver a form, document, or packet. |
| **Grow** | Kinderly's Kanban pipeline for tracking prospective families. |
| **Manage** | Kinderly's full CCMS add-on (children, staff, classrooms, billing, etc.). |
| **Ratio band** | A time-based child-to-staff ratio requirement for a classroom. |
| **Billing account** | A Manage object linked to guardians and children that tracks charges and payments. |
| **Billing run** | A bulk generation of bills across every active billing account. |
| **KinderConnect** | The KinderSystems integration used to report attendance for state subsidy billing. |
